git-Распределенная система версии 
это система для отслеживания и введения истории изменения файлов в вашем проекте

Репозиторий- Хранилище вашего кода и истории его изменений 

GitHub- Веб сайт для резмещения git -репозиториев в совместной разработке проектов 

Команда Git
1. git init -она создаёт новый локальный гит репозиторий на ПК в той директории в которой была прописана команда.
2. git add -команда для добавления файлов в ситстем у отслеживания Гита , после этой команды изменения начинают отслеживаться Гитом 
git commit -m '<commit>' -как только достигаем определенного момента в разработке, то тогда мы сохраняем и коментируем изменения в локальной репозиторий. 
3. git remote add -это команда для того чтобы связать наш локальный репозиторий с удаленным репозиторием, то есть добавляет подлючепние к репозиторий на гитхабе.
       git remote add <name: origin> <ссылка на репо: ssh>
git push -команда для отправки все изменгения на локальных репозиториях в удаленный репозиторий
       git push origin <branch(main)>
